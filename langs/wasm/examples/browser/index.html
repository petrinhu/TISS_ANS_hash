tiss-hash-wasm

Calcula o hash MD5 do epílogo <ans:hash> de um XML do Padrão TISS/ANS, inteiramente dentro desta aba.

Privacidade (LGPD): o arquivo XML é lido e hasheado localmente, no seu navegador, via WebAssembly. Nada é enviado a nenhum servidor: nem o XML (que pode conter dados de paciente), nem o hash. Você pode desconectar a internet depois de carregar a página que continua funcionando.

Selecione um arquivo XML TISS para calcular o hash.

Como servir localmente (o navegador exige HTTP, não file://, para carregar o módulo WASM): 

cd langs/wasm
python3 -m http.server 8000
# abra http://localhost:8000/examples/browser/
